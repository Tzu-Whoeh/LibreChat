# 业务约束

> 用用户能感受到的语言描述要求；架构 agent 会将其翻译为技术方案。

## 性能 / 容量

- 客服团队约 **20 人** 同时在线使用。
- _待补充：高峰期同时进行的会话数、可接受的页面/搜索响应时间。_

## 安全 / 合规

_待补充：例如「个人信息保护」「数据不能出境」「需要审计日志」。_

## 数据集成

系统需要从以下现有系统拉取/同步数据：
- **CRM**：客户基础资料
- **ERP**：订单信息
- **邮箱系统**：历史邮件 + 邮件渠道收发
- **知识库**：标准答案与文章

咨询渠道接入：
- **电话**（_待澄清：是否接入现有呼叫中心？_）
- **邮件**
- **网站在线聊天**

## 访问范围 / 权限

_待补充：谁能看到什么、谁能操作什么。_

## 业务规则

_待补充：例如「工单 24 小时未响应自动升级」。_
