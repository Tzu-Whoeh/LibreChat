# 业务约束

> 用用户能感受到的语言描述要求；架构 agent 会将其翻译为技术方案。

## 性能 / 容量

_待补充：例如「高峰期 X 人同时在线，响应小于 X 秒」。_

## 安全 / 合规

_待补充：例如「个人信息保护」「数据不能出境」「需要审计日志」。_

## 数据集成

_待补充：例如「员工数据从 HR 系统拿」「订单数据来自 ERP」。_

## 访问范围 / 权限

_待补充：谁能看到什么、谁能操作什么。_

## 业务规则

_待补充：例如「工单 24 小时未响应自动升级」。_
