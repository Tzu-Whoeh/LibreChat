# UX 说明

> 供原型 agent 使用的界面与交互参考。用产品经理与客服能理解的语言描述。

## 平台
- **客服端**：仅 PC 网页端（Chrome / Edge 现代浏览器）
- **客服主管端**：与客服端同一系统，按角色权限切换可见模块
- **不做**：手机 App、平板适配

## 整体风格
- 偏专业、信息密度高的工作台风格（参考企业 SaaS 工作台，如飞书工作台、Zendesk Agent Workspace）
- 浅色主题为主
- 关键提示（敏感拦截、AI 推荐）用明显色块/图标强调

## 主要页面

### 1. 对话工作台（核心页）
- **左栏**：会话列表（网页 + 微信混合），按未回复优先排序，标识渠道图标
- **中栏**：当前会话消息流，类似 IM 风格
- **右栏**：AI 推荐答案卡片
  - 卡片内含：建议回复文本、来源出处（点击可跳转知识库/SharePoint 链接）、3 个操作按钮：**直接发送** / **编辑后发送** / **忽略**
  - 多条推荐时叠放，可切换
- **敏感命中**：右栏 AI 卡片替换为红色提示条："此对话命中敏感规则，请人工处理"

### 2. 电话工单页
- 上传录音区（拖拽/点击）
- 上传后展示：转写文字、AI 摘要（问题/处理结果/分类标签）
- 可编辑摘要字段
- 「保存并归档」按钮

### 3. 知识库管理（主管）
- FAQ 列表，支持分类树、搜索、启用/停用开关
- 增加 / 编辑 / 删除按钮
- 变更后显示"已同步给 AI"状态

### 4. 敏感规则管理（主管）
- 规则列表（关键词或表达式）
- 增删改、启停
- 命中测试输入框（输入一段话试试看会不会命中）

### 5. 周报页（主管）
- 报告日期选择
- 热点话题 Top10 + 趋势图
- 每个话题可展开看典型案例
- 「导出 PDF」按钮

### 6. 数据看板（主管）
- 三大核心指标卡：响应时长、CSAT、AI 采纳率
- 按周/月切换
- 渠道分组对比

### 7. 问题库
- 全渠道问题列表
- 四维筛选：渠道 / 用户 / 话题 / 时间

## 关键交互

- **AI 推荐响应慢时**：右栏显示 loading 动画，超过 5s 提示「AI 正在思考」
- **客服结束会话**：点击「结束会话」按钮，弹出确认；触发 CSAT 邀评
- **用户超时**：默认 10 分钟无消息自动结束（具体阈值待定）
- **CSAT 评价**：用户端弹出 1-5 星打分 + 可选文字反馈
